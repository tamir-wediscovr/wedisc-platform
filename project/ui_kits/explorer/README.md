# Explorer UI Kit

weDISCOVR Explorer — iOS super-app, iPhone 16 Pro, **402×874px (locked)**.

## Files

- `index.html` — click-thru prototype. Home → Drops → Event → Caught Gift flow.
- `App.jsx` — screen router + shell.
- `StatusBar.jsx` — 48px dark status bar.
- `TabBar.jsx` — bottom nav (Home · Discover · Hunt · Passport · You).
- `Home.jsx` — feed screen (header, moments rail, hero, happening now, curators).
- `Drops.jsx` — Drops list (DISC pill, caught/available tabs).
- `EventDetail.jsx` — event page with About / Schedule / Curator / Who's going.
- `CaughtGift.jsx` — claimed-gift reward screen.
- `Primitives.jsx` — Pill, GlassCard, Avatar, Divider, CTA.
